# Hacker系统之域名防封杀

背景：由于公司产品的形态导致域名会在微信下进行封杀，或者由于备案过期等原因导致部分域名失效。不及时更换域名会对公司的收入产生极大的影响。

## 现状
推啊的域名在失效后可以得到及时的替换，并以钉钉告警的方式告知所要提醒的人。

## 正在做的
### 1、域名白名单管理。
白名单运营可配置，活动前端通过配置的白名单及自己的域名去判断是否拦截。

### 2、域名屏蔽监控
#### A.域名屏蔽监控虚拟化
现有的微信域名屏蔽监控是在真机上跑，会有各种不可控的因素去影响域名屏蔽监控的有效性。例如：断电、断网等。
